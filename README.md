# Middleware

Open the new tab in terminal and follow the instructions below

1. Move to the middleware directory using the command.

```terminal
# cd middleware
```

2. Install node modules using the command.

```terminal
# npm install
```

3. Run the middleware using the command.

```terminal
# npm start
```
